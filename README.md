This project contains a custom Calendar component built inside App.js.
The calendar renders a full month view based on a date passed as a prop.
Features:
Displays a complete monthly calendar,
Accepts any JavaScript Date object as input,
Automatically adapts to:
        Month length, Leap years, Weekday alignment, Easy to customize and extend
        Fully functional UI for month display.
To show another month, simply modify: exampleDate constant
